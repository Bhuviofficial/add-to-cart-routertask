# Add to Cart Router

A React application demonstrating routing and cart management using React Router. This project is part of the FSD-MERN curriculum and showcases how to build a simple e-commerce cart system with navigation.

## Features

- Product listing page
- Product detail view
- Add to cart functionality
- Cart page with item management
- Navigation using React Router v6+
- State management using React hooks

## Technologies Used

- React
- React Router DOM
- JavaScript (ES6+)
- CSS (or your preferred styling method)

## Getting Started

### Prerequisites

- Node.js (v14 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Bhuviofficial/add-to-cart-router
    cd add-to-cart-router
    ```

2. Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```

### Running the App

```bash
npm start
# or
yarn start
```

The app will be available at [http://localhost:3000](http://localhost:3000).

## Usage

- Browse products on the home page.
- Click on a product to view details.
- Add products to your cart.
- View and manage your cart from the cart page.

## Customization

- Add more products by updating the product data source.
- Enhance cart features (e.g., quantity update, remove items).
- Integrate with a backend or local storage for persistence.
